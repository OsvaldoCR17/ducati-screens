# Ducati Screens

Un proyecto de desarrollo de pantallas interactivas diseñado para capturar la esencia de la marca Ducati.

### Detalles

- **Lenguaje principal:** HTML (45.6% del proyecto).  
- **Interactividad:** Construido con TypeScript para añadir funcionalidades dinámicas y robustas.  
- **Estilo y diseño:** SCSS para estilizar y brindar un aspecto profesional y moderno a la interfaz.  
- **Objetivo:** Mostrar la potencia y elegancia de Ducati a través de pantallas interactivas de vanguardia.

### Características principales

- Desarrollo de interfaces integradas con animaciones modernas.  
- Enfoque en la experiencia del usuario (UX) y diseño adaptable.  
- Estructura de código modular y reutilizable.  
- Compatible con diferentes dispositivos y plataformas.
